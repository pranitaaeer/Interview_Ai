# 🤖 Interview_AI - Full Stack AI Interview Preparation Platform

Radhe Radhe 🙏  
I am Pranita Aeer

This is a full-stack AI-powered interview preparation platform built using MERN stack with AI integration.

---

## 🚀 Project Overview

Interview_AI helps users prepare for interviews using AI. It provides:

- Resume upload & analysis
- AI-generated interview questions
- Resume scoring system
- Weekly learning roadmap
- User authentication system
- Chatbot AI assistant (optional module)

---

## 🧠 Features

### 👤 User Side
- Signup / Login (JWT authentication)
- Upload resume (PDF/Text)
- Get AI feedback on resume
- Generate interview questions
- Personalized roadmap

### 🤖 AI Features
- Resume analysis using AI (Gemini / Groq)
- Smart interview question generation
- Skill-based recommendations

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Redux Toolkit
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT Authentication
- Google Gemini / Groq AI API
